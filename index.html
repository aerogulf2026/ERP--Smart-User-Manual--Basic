Introduction

Welcome to the ERP Interactive Manager. This application is designed to provide a real-time, interactive platform for executing Emergency Response Plan (ERP) and Emergency Action Plan (EAP) checklists. It ensures that all roles are covered, actions are tracked, and communication is streamlined during an emergency or exercise.

Logging In & User Identity

Upon starting the application, you will be prompted to identify yourself. This is a critical step for accountability and tracking.

  • Enter Your Details: You must enter your name or employee number in the first field.
  • Select Your Role: Choose the role that reflects your function in the current emergency or drill from the dropdown menu.
  • User Association: All your actions (completing tasks, adding notes) will be logged against your name and role.

How to Use the Application

  • Select a Checklist: Use the sidebar to find and select the checklist for a specific role.
  • Complete Tasks: Mark tasks complete to log them with a timestamp.
  • Add Notes & Fill Forms: Use the text area below each task to add relevant notes.
  • Join the CMC Video Conference: Use the red button for immediate 24/7 access to the Google Meet conference.

Roles & Permissions

Your ability to interact with checklists is determined by the role you select at login.

Standard

Can only edit checklists that match their job function.

Power User

Accountable/Safety Manager — elevated permissions to edit any task on any checklist.

Read-Only

DP Representative — for observation purposes only; cannot make changes.

Administrator

Safety Manager — can reset all application data after an exercise is concluded.

Frequently Asked Questions (FAQ)

What happens if I go offline?

The app works offline. Changes are saved to your device and sync automatically once you are back online. A yellow banner will appear when offline.

Can I use this on my phone or tablet?

Yes, the application is fully responsive and works on desktops, tablets, and smartphones.

Who can see the notes and task updates I make?

All data is synced in real-time. Any user with access can see the updates, ensuring team-wide situational awareness.
